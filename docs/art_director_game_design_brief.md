# 아트디렉터 전달용 전체 게임 기획안

작성일: 2026-05-26  
문서 목적: 아트디렉터가 게임의 핵심 경험, 세계관, 캐릭터, UI, 에셋 우선순위를 한 번에 파악하고 아트 방향을 제안할 수 있도록 정리한 브리프.

---

## 1. 게임 한 줄 설명

중세 판타지 세계에서 균형형 주인공이 길 위의 동료를 영입하고 강화하며 3막을 돌파하는, **동료 시스템 중심의 싱글플레이 2D 로그라이크 덱빌딩 게임**.

핵심은 "강한 유물을 먹어서 이겼다"가 아니라, "이번 런에서 만난 동료와 장비 조합 때문에 전술 부대가 달라졌다"는 감각이다.

```txt
장르: 2D 로그라이크 덱빌딩
플랫폼: PC + 모바일
화면 방향: 가로 화면
엔진: Godot 4.x
과금: 게임 구매형
멀티플레이: 없음
아트 방향: 전문가 제작 예정
```

---

## 2. 아트가 전달해야 할 핵심 경험

이 게임의 구조는 일반 덱빌더의 `카드 + 유물 + 포션 + 이벤트`가 아니다.

```txt
본 게임의 축: 카드 + 동료 + 장비 + 이벤트 + 여관
```

아트가 가장 강하게 보여줘야 하는 것은 아래 4가지다.

```txt
1. 동료가 이번 런의 얼굴이 된다.
2. 주인공과 동료들이 길 위에서 임시 전술 부대를 이룬다.
3. 장비는 유물처럼 무제한 쌓이는 보상이 아니라, 누가 착용하느냐가 중요한 물건이다.
4. 여관, 상점, 이벤트는 단순 메뉴가 아니라 위험한 길 위의 장소다.
```

플레이어가 스크린샷만 봐도 알아야 하는 것:

```txt
이 게임은 카드 게임이다.
동료가 전투와 런 방향을 바꾼다.
중세 판타지지만 영웅 왕국 서사보다 길 위의 생존과 동료애에 가깝다.
UI는 모바일 가로 화면에서도 읽히는 전술 인터페이스다.
```

## 3. 세계관 방향

### 기본 세계관

무너진 변경 왕국의 길을 따라, 주인공이 동료를 모으며 왕성으로 향하는 로드 판타지.

```txt
장르 정서: 중세 판타지 + 전술 부대 + 길 위의 생존
배경 키워드: 서약, 변경길, 무너진 왕국, 낡은 장비, 위험한 여관, 수상한 상점, 동료의 흔적
주인공 설정: 몰락한 왕국의 전술 장교 또는 호위대장
```

왕국을 구하는 거대한 영웅담보다, 끝까지 같이 살아남는 작은 부대의 이야기다.

### 유지해야 할 정서

```txt
낡았지만 살아 있는 세계
위험하지만 완전히 절망적이지 않은 여정
반짝이는 왕궁보다 먼지 묻은 길과 전장 흔적
차갑고 잔혹한 전쟁보다 동료, 약속, 임시 거처의 온기
```

### 피해야 할 정서

```txt
현대적인 네온 판타지
과도하게 화려한 모바일 수집형 RPG 느낌
너무 귀엽거나 장난감 같은 SD 비율
피와 공포를 앞세우는 다크 호러
캐릭터보다 장비와 이펙트만 튀는 그림
```

## 4. 비주얼 디자인 기둥

### 4.1 전술적 가독성

전투는 카드, 적 의도, 동료 기본 공격, 전술 표식이 동시에 보인다. 화면이 아름다워도 읽히지 않으면 실패다.

```txt
적 실루엣은 역할별로 분명해야 한다.
동료는 전투 화면에서 작은 초상만으로도 구분되어야 한다.
상태 효과와 의도 아이콘은 텍스트 없이도 1차 의미가 보여야 한다.
전술 표식은 턴 중 가장 중요한 시선 유도 장치다.
```

### 4.2 동료 중심의 런 정체성

동료는 숫자 보너스가 아니라 "이번 런의 빌드 문"이다.

```txt
로완을 고르면 표식과 취약의 런처럼 보인다.
세라를 고르면 빠른 연타와 얇은 단검의 런처럼 보인다.
엘드릭을 고르면 방패와 버티기의 런처럼 보인다.
```

동료마다 실루엣, 주 색, 무기, 장비 궁합, 카드 프레임 포인트가 달라야 한다.

### 4.3 길 위의 물성

세계는 손에 잡히는 재료로 구성된다.

```txt
낡은 금속
해진 천
가죽 끈
목재 간판
양피지 지도
밀랍 봉인
찌그러진 동전
그을린 돌
약초와 유리병
```

UI도 완전한 현대 평면 UI보다는, 전술 지도/카드/장비함의 물성을 얇게 품는 쪽이 어울린다. 단, 장식이 텍스트 가독성을 해치면 안 된다.

### 4.4 따뜻한 위험

어두운 판타지이되, 화면 전체가 검거나 칙칙하면 안 된다. 위험한 지역 안에도 불빛, 천 색, 금속 반사, 마법 색 같은 시선 포인트가 있어야 한다.

## 5. 전체 아트 스타일 제안

### 추천 방향

```txt
2D 일러스트 기반
반실사보다 스타일라이즈된 중세 판타지
선과 실루엣이 분명한 캐릭터
카드 일러스트는 회화적이되 작은 크기에서도 읽히는 구성
UI는 장식보다 정보 계층 우선
```

캐릭터 비율은 현실적인 6.5~7.5등신을 기본으로 한다. 너무 실사적이면 카드 게임의 조작감이 무거워지고, 너무 SD에 가까우면 로드 판타지의 생존감이 약해진다.

### 색채 원칙

전체 게임이 한 가지 색만으로 보이면 안 된다. Act별로 주조색은 다르되, 항상 보조색과 시선 유도색을 함께 둔다.

| 구간 | 기본 인상 | 주 색감 | 보조/강조 |
|---|---|---|---|
| Act 1 무너진 변경길 | 먼지, 녹슨 철, 해질녘 길 | 회녹색, 흙빛, 낡은 철 | 붉은 깃발, 모닥불 주황, 푸른 그림자 |
| Act 2 오염이 번진 접경지 | 숲, 폐광, 타락한 기사단 | 짙은 초록, 검은 목재, 탁한 보라 | 약초 초록, 독 연기, 은색 문장 |
| Act 3 검은 관문 | 왕성으로 가는 최종 압박 | 검은 돌, 차가운 회색, 백색 안개 | 진홍, 낡은 금, 성스러운 백색 |
| UI 공통 | 전술 지도와 장비함 | 양피지, 먹색, 금속 회색 | 역할별 색상과 상태 아이콘 |

## 6. 화면 구성 방향

### 전투 화면

권장 구도:

```txt
좌측: 주인공과 보유 동료
우측: 적
하단: 손패 카드
상단 또는 측면: 체력, 골드, 남은 에너지, 덱/버림/소멸 더미
적 위: 의도 아이콘
대상 위: 전술 표식 아이콘
```

동료는 별도 체력이 없지만 화면에 존재해야 한다. 초상화만으로 끝나면 동료 시스템의 감정적 힘이 약해진다.

가능한 표현:

```txt
전투 좌측에 작은 스탠딩 실루엣 배치
하단 또는 좌측에 동료 초상 + 유대 단계 + 장비 슬롯 요약
턴 종료 후 동료 기본 공격 시 짧은 컷인 또는 실루엣 액션
```

### 맵 화면

맵은 갈림길이 있는 로드 판타지의 핵심 화면이다.

```txt
양피지 또는 전술 지도 느낌
노드는 길 위의 장소 아이콘으로 표현
보스 방향은 항상 시각적으로 압박감이 있어야 함
Act 1 중반에는 동료의 흔적 노드가 특별히 눈에 띄어야 함
```

### 상점 화면

상점은 안전한 메뉴가 아니라, 길 위의 낡은 거래 장소다.

```txt
낡은 카운터
걸린 장비
카드 묶음
서비스 표지판
상인의 손과 동전
```

상점 UI 목표는 "전부 사고 싶지만 하나나 둘만 살 수 있다"는 경제 압박을 보여주는 것이다.

### 여관 화면

여관은 회복 노드이면서 이벤트 공간이다. 일반 여관과 이벤트 여관이 첫눈에 구분되어야 한다.

```txt
일반 여관: 정직하고 따뜻함, 가격과 효과 명확
이벤트 여관: 수상하지만 치명적 함정은 아님, 이상한 방 이름과 묘한 조명
```

이벤트 여관은 공포가 아니라 "수상한데 혹시 좋은 일이 생길지도 모른다"는 쪽이다.

## 7. 주인공 아트 방향

주인공은 동료 시스템을 받쳐주는 균형형 지휘관이다. 너무 튀면 동료가 흐려지고, 너무 밋밋하면 전투 화면의 중심이 약해진다.

```txt
역할: 몰락한 왕국의 전술 장교 또는 호위대장
무기: 검, 지휘봉, 짧은 창 중 택1 권장
실루엣: 망토 또는 반망토, 실용적인 갑옷, 지도/명령서 소지
성격: 침착함, 피로감, 책임감
색감: 중립 금속색 + 낡은 천 + 작은 왕국 문장색
```

피해야 할 것:

```txt
왕관을 쓴 영웅왕
과도하게 거대한 무기
동료보다 화려한 갑옷
직업이 모호한 평범한 모험가
```

## 8. 동료 10명 아트 방향

동료는 각자 전투 역할, 성격, 장비 궁합이 보이는 캐릭터여야 한다.

| 동료 | 역할 | 비주얼 핵심 | 색/소재 | 카드/VFX 힌트 |
|---|---|---|---|---|
| 붉은창 로완 | 전술 표식 단일 딜러 | 붉은 망토, 긴 창, 표적을 보는 자세 | 진홍 천, 어두운 철, 깃발 | 직선 찌르기, 붉은 표식, 취약 균열 |
| 검은매 세라 | 0비용 연타 암살자 | 짧은 검 두 자루, 가벼운 척후병 복장 | 검은 가죽, 청록 포인트, 은색 칼날 | 빠른 잔상, 얇은 베기 선, 암호 표식 |
| 화약장이 브람 | 광역 폭발 기술자 | 폭약 주머니, 그을린 장갑, 낮고 단단한 체형 | 그을음, 황동, 주황 불씨 | 폭발 원형, 파편, 연막 |
| 방패기사 엘드릭 | 안정 방어 기사 | 낡은 문장 방패, 중갑, 곧은 자세 | 회색 철, 낡은 금장, 푸른 천 | 방패광, 반격 충격파, 맹세 문장 |
| 수도승 마렌 | 회복/지속 생존 | 회색 수도복, 철제 지팡이, 약초 주머니 | 회색 천, 말린 약초, 따뜻한 빛 | 작은 기도문, 낮은 회복광, 호흡 이펙트 |
| 철갑 곰지기 토르 | 피해 흡수/약화 | 큰 가죽 갑옷, 북방 전사, 무거운 체중감 | 짙은 가죽, 털, 차가운 철 | 둔탁한 충격, 서리 입김, 약화 포효 |
| 독초술사 리나 | 독/디버프 | 초록 망토, 유리병, 약초 가방 | 초록 유리, 갈색 가죽, 노란 독액 | 독 방울, 연무, 부식 흔적 |
| 별읽는 노아 | 드로우/손패 조작 | 별무늬 로브, 카드/천문 도구 | 남색 천, 은색 별, 잉크 | 별선, 카드 궤도, 예지 원 |
| 백사제 이솔 | 보호/위기 회복 | 흰 사제복, 작은 은종, 성스러운 천 | 백색 천, 은, 따뜻한 금빛 | 종 파동, 보호막, 부드러운 회복광 |
| 은주머니 카일 | 골드/상점 보상 | 웃는 상인, 동전 주머니, 겹겹의 옷 | 노란 금속, 짙은 초록 천, 갈색 가죽 | 동전 튐, 거래 도장, 계산서 |

### MVP 우선 동료

첫 구현과 아트 테스트는 아래 3명으로 한다.

```txt
로완: 동료 시스템의 표식 조종감을 보여주는 기준 캐릭터
세라: 빠른 템포와 카드 회전의 재미를 보여주는 캐릭터
엘드릭: 생존형 동료도 매력적이어야 함을 증명하는 캐릭터
```

이 3명은 실루엣, 색상, 전투 감각이 명확히 달라야 한다.

## 9. 적과 보스 방향

### 적 공통 원칙

```txt
적은 역할이 실루엣으로 보여야 한다.
일반 적은 과하게 복잡하지 않게 만든다.
엘리트는 한눈에 위험도가 높아 보여야 한다.
보스는 Act의 테마를 시각적으로 대표한다.
```

적 의도와 애니메이션이 읽혀야 하므로, 장식이 너무 많거나 배경과 색이 붙으면 안 된다.

### Act 1: 무너진 변경길

```txt
일반 적: 산적, 탈영병, 굶주린 짐승, 초소 잔당
엘리트: 무거운 갑옷의 탈영 정예, 매복 궁수단, 버려진 초소 수문장
보스: 무너진 성문의 탈영대장
```

Act 1 보스는 "동료를 만나기 전 마지막 혼자서의 시험"이다. 방패, 깃발, 부서진 문장, 지친 병사 느낌이 중요하다.

### Act 2: 오염이 번진 접경지

```txt
일반 적: 숲의 이단자, 폐광 노동자 망령, 타락한 기사단 하급병, 수상한 상단 경호원
엘리트: 마녀 숲의 감시자, 폐광 거대 장치, 검은 맹세 기사
보스: 쌍서약 기사단
```

Act 2는 첫 동료와의 시너지가 본격화되는 구간이다. 적도 단일 해법보다 방어/공격/디버프 대응을 요구하는 형태로 보이면 좋다.

### Act 3: 왕성으로 가는 검은 관문

```txt
일반 적: 최종 군세, 왕성 친위대 잔당, 검은 관문 수비병
엘리트: 문장 없는 기사, 왕성 마도관, 서약 파괴자
보스: 검은 관문의 섭정
```

Act 3 보스는 주인공 덱, 동료 2명, 장비 조합의 최종 검증이다. 단순히 큰 괴물이 아니라, 왕국 몰락과 서약 파괴의 얼굴이어야 한다.

## 11. 카드와 UI 아트 방향

### 카드 프레임

카드는 이 게임의 가장 많이 보는 오브젝트다. 작은 화면에서도 비용, 타입, 희귀도, 소유자가 보여야 한다.

필수 구분:

```txt
소유자: 주인공 카드 / 동료 카드
타입: 공격 / 스킬 / 파워
희귀도: starter / common / uncommon / rare
비용: 좌상단 또는 가장 빠르게 읽히는 위치
강화 여부: 카드명 또는 프레임에 명확히 표시
```

동료 카드는 해당 동료의 색과 문양이 프레임 일부에 들어가야 한다. 단, 프레임이 너무 강하면 카드 일러스트와 텍스트가 묻힌다.

### 카드 일러스트

카드 일러스트는 전체 풀 기준으로 많다.

```txt
주인공 카드: 40장
동료 카드: 10명 x 8장 = 80장
```

초기에는 모든 카드에 풀 일러스트를 넣기보다, 카드 타입별/동료별 공통 모티프를 활용한 제작 효율안을 고려할 수 있다.

권장 접근:

```txt
MVP: 핵심 카드만 개별 일러스트, 나머지는 아이콘/모티프 기반 임시 아트 가능
정식: 주인공 주요 카드와 동료 카드 중심으로 개별 일러스트 확장
```

### 주요 UI 아이콘

```txt
에너지
체력
골드
방어도
카드 드로우
버림/소멸
전술 표식
취약
약화
독
회복
장비 슬롯: 투구 / 갑옷 / 무기
노드: 전투 / 엘리트 / 보스 / 상점 / 여관 / 이벤트 / 보물 / 동료 흔적
```

### 전술 표식

전술 표식은 동료 공격 대상을 조종하는 핵심 시각 요소다.

```txt
아이콘 후보: 작은 깃발, 화살표, 붉은 표적핀
표시 위치: 적 머리 위 또는 발밑 원형 마커
요구사항: 모바일에서도 즉시 보일 것
```

표식은 공격적이어야 하지만, 독/취약 같은 상태 효과와 혼동되면 안 된다.

### 유대 단계

동료 강화 상태는 `유대 0 / 1 / 2`로 표현한다.

```txt
유대 0: 영입 직후
유대 1: 해당 동료 방향을 밀고 있음
유대 2: 런의 중심 동료가 됨
```

UI 표현 후보:

```txt
초상화 주변 매듭
작은 서약 문장 0~2개
장비 슬롯 근처의 빛나는 봉인
```

## 12. 장비 아트 방향

장비는 유물 대체 시스템이지만, 무제한 수집품이 아니다. 누가 착용하느냐가 중요하다.

장비 슬롯:

```txt
투구
갑옷
무기
```

캐릭터별 최대 3슬롯이며, 주인공 + 동료 2명 기준 최대 9슬롯이다.

장비 아이콘은 아래를 만족해야 한다.

```txt
작은 크기에서도 슬롯이 구분된다.
장비 희귀도가 보인다.
팀 전체 효과와 장착자 전용 효과가 구분된다.
동료 궁합을 암시하는 소재나 색이 있다.
```

장비 예시:

```txt
선봉대 철투구: 모든 동료 기본 공격 피해 +1
전술가의 모자: 주인공 첫 턴 드로우 +1
치유사의 은관: 회복 효과 +20%
지휘검: 모든 공격 카드 피해 +1
은 상단 배지: 상점 카드 가격 10% 감소
수상한 열쇠: 이벤트 선택지에서 보물 결과 확률 증가
```

## 13. VFX와 모션 방향

전투 애니메이션은 화려함보다 리듬과 판독성이 우선이다.

### 기본 원칙

```txt
카드 사용 후 0.2~0.5초 안에 결과가 읽혀야 한다.
동료 기본 공격은 자동으로 발생하므로 짧고 명확해야 한다.
피해, 방어, 회복, 디버프의 색상 언어가 일관되어야 한다.
모바일 성능을 고려해 과한 파티클을 피한다.
```

### 색상 언어 예시

| 효과 | 권장 색/형태 |
|---|---|
| 물리 피해 | 밝은 금속 섬광, 짧은 충격선 |
| 방어도 | 청회색 방패광, 금속음 |
| 회복 | 따뜻한 백색/금빛, 낮은 파동 |
| 독 | 탁한 초록 방울/연무 |
| 약화 | 무거운 회색 보라, 흐려지는 실루엣 |
| 취약 | 붉은 균열, 깨지는 방패선 |
| 드로우/예지 | 잉크선, 별점, 종이 넘김 |
| 골드 | 작고 짧은 동전 반짝임 |
| 전술 표식 | 깃발/핀 고정, 짧은 지휘선 |

## 17. 아트디렉터에게 먼저 받고 싶은 제안

초기 미팅에서 아래 항목을 먼저 결정하면 제작 리스크가 줄어든다.

```txt
1. 캐릭터 비율과 렌더링 밀도
2. 카드 일러스트를 전부 개별 제작할지, 모티프/아이콘 혼합으로 갈지
3. 전투 화면에서 동료를 스탠딩으로 보여줄지, 초상 중심으로 보여줄지
4. UI 물성의 정도: 양피지/금속 장식과 현대적 가독성의 비율
5. Act별 팔레트와 키 비주얼
6. MVP 기준으로 어디까지 완성 아트가 필요하고 어디까지 임시 아트가 허용되는지
```

가장 중요한 첫 결정은 **동료가 전투 화면에서 얼마나 물리적으로 존재하는가**다. 이 결정이 캐릭터 제작량, 애니메이션, UI 레이아웃, 카드 보상 화면의 감정선을 모두 좌우한다.
