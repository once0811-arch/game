# 아트디렉터 전달용 전체 게임 기획안

작성일: 2026-05-26  
문서 목적: 아트디렉터가 게임의 핵심 경험, 세계관, 캐릭터, UI, 에셋 우선순위를 한 번에 파악하고 아트 방향을 제안할 수 있도록 정리한 브리프.

---

## 1. 게임 한 줄 설명

마법 재난으로 붕괴 중인 중세 판타지 세계에서 균형형 주인공이 길 위의 동료와 서약 전술을 맺고, 유대 점수를 쌓으며 3막을 돌파하는 **동료 시스템 중심의 싱글플레이 2D 로그라이크 덱빌딩 게임**.

핵심은 "이번 런에서 만난 동료, 선택한 서약 전술, 장비 조합 때문에 전술 부대가 달라졌다"는 감각이다.

```txt
장르: 2D 로그라이크 덱빌딩
플랫폼: PC + 모바일
화면 방향: 가로 화면
엔진: Godot 4.x
과금: 게임 구매형
멀티플레이: 없음
아트 방향: 전문가 제작 예정
```

---

## 2. 아트가 전달해야 할 핵심 경험

이 게임의 구조는 카드 보상만 반복하는 덱빌더가 아니다.

```txt
본 게임의 축: 카드 + 동료 + 서약 전술 + 유대 점수 + 장비 + 이벤트 + 여관
```

아트가 가장 강하게 보여줘야 하는 것은 아래 5가지다.

```txt
1. 동료가 이번 런의 얼굴이 된다.
2. 주인공과 동료들이 길 위에서 임시 전술 부대를 이룬다.
3. 서약 전술은 동료마다 다른 전투 리듬과 증표 조항을 만든다.
4. 장비는 누가 착용하느냐가 중요한 물건이다.
5. 여관, 상점, 이벤트는 단순 메뉴가 아니라 위험한 길 위의 장소다.
```

플레이어가 스크린샷만 봐도 알아야 하는 것:

```txt
이 게임은 카드 게임이다.
동료가 전투와 런 방향을 바꾼다.
선택한 서약 전술과 유대 점수가 동료와의 관계를 보여준다.
중세 판타지지만 영웅 왕국 서사보다 괴물 재난 속 생존과 동료 계약에 가깝다.
UI는 모바일 가로 화면에서도 읽히는 전술 인터페이스다.
```

### 2.1 게임 시스템 한눈에 보기

아트디렉터는 전체 시스템을 아래 흐름으로 이해하면 된다.

```txt
런 시작
→ 맵에서 다음 노드 선택
→ 전투 / 이벤트 / 상점 / 여관 / 보물 / 엘리트 진행
→ 카드, 골드, 장비, 회복, 강화, 동료 관련 보상 선택
→ 다음 노드 선택 반복
→ Act 보스전
→ Act 1/2 보스 후 동료 영입 또는 기존 동료 유대 보상
→ Act 3 버려진 고대 성의 코어 보스 처치 시 런 클리어
```

이 게임은 한 장면의 일러스트보다, **반복해서 보는 시스템 화면의 판독성**이 중요하다. 전투, 보상, 맵, 동료 영입, 장비, 상점, 여관 화면이 모두 같은 세계의 물성을 공유해야 한다.

### 2.2 전투 시스템 요약

기본 전투는 아래 순서로 돈다.

```txt
턴 시작: 카드 6장 드로우, 에너지 4 획득
플레이어 행동: 카드 사용, 적 대상 지정, 전술 표식 갱신
서약 전술 검사: 선택한 동료 서약 조건을 만족하면 발동
턴 종료: 동료들이 전술 표식 대상 또는 우선순위 대상 기본 공격
적 턴: 적 의도에 따라 공격/방어/디버프
반복
```

전투 화면에서 가장 먼저 읽혀야 하는 정보:

| 정보 | 시스템 의미 | 아트/UI 요구 |
|---|---|---|
| 손패 카드 | 이번 턴 선택지 | 비용, 타입, 소유자, 강화 여부가 작게도 읽혀야 함 |
| 에너지 4 | 이번 턴 행동 예산 | 숫자와 남은 행동감이 즉시 보여야 함 |
| 적 의도 | 다음 피해/방어/디버프 예고 | 실루엣과 아이콘이 배경에 묻히면 안 됨 |
| 전술 표식 | 동료 기본 공격 우선 대상 | 상태이상과 혼동되지 않는 별도 표식 필요 |
| 동료 패널 | 보유 동료, 서약 전술, 유대 점수, 장비 | 동료가 장식이 아니라 전투 시스템임을 보여야 함 |
| 서약 발동 | 선택한 전투 조항의 조건 충족 | 컷신급 과장보다 짧고 반복 가능한 반응이 필요 |

### 2.3 런 정체성을 만드는 시스템

플레이어의 이번 런은 아래 조합으로 달라진다.

```txt
주인공 덱
+ 동료 1의 서약 전술
+ 동료 1의 전용 카드와 유대 점수
+ 동료 2의 서약 전술
+ 동료 2의 전용 카드와 유대 점수
+ 주인공/동료 장비 슬롯
+ 상점/여관/이벤트 선택
```

아트가 시스템을 이해하고 보여줘야 하는 지점:

| 시스템 | 플레이어가 하는 선택 | 시각적으로 보여야 하는 것 |
|---|---|---|
| 동료 영입 | 후보 중 1명을 선택 | 이 동료를 고르면 런의 얼굴과 전투 리듬이 바뀐다는 감각 |
| 서약 전술 | 동료별 3개 중 1개 선택 | 증표에 적는 전투 조항, 선택 후 런 동안 고정되는 무게 |
| 유대 점수 | 전투/이벤트를 함께 겪으며 0~100 누적 | 30/60/100 구간이 작지만 기분 좋게 보이는 관계 성장 |
| 동료 카드 | 보유 동료 카드가 보상 후보에 섞임 | 주인공 카드와 동료 카드의 소유자 구분 |
| 장비 | 주인공/동료 슬롯에 장착 | 누가 착용하느냐가 중요한 물건이라는 인상 |
| 상점 | 카드/장비/서비스를 골드로 구매 | 사고 싶은 것이 많지만 골드가 부족한 압박 |
| 여관 | 회복과 이벤트성 방 선택 | 안전한 회복과 수상한 기회가 구분되어야 함 |
| 이벤트 | 위험을 알고 선택 | 숫자 보상만이 아니라 길 위의 사건처럼 보여야 함 |

### 2.4 보상과 경제 구조

카드 보상은 기본적으로 3장 중 선택하며, 필요 없으면 스킵해서 소량 골드를 얻는다. 보유 동료가 생기면 동료 전용 카드가 보상 후보에 섞인다.

골드는 단순 재화가 아니라 생존 자원이다.

```txt
상점: 카드, 장비, 강화, 제거, 복제, 변화 구매
여관: 회복과 방 선택
이벤트: 비용 지불, 위험 감수, 보상 획득
장비: 제한된 슬롯을 누구에게 줄지 결정
```

따라서 상점/여관/보상 화면은 메뉴처럼 보이면 안 된다. 플레이어가 다음 전투를 계산하며 선택하는 **전술 준비 공간**처럼 보여야 한다.

### 2.5 본 게임에 없는 시스템

아트 방향에서도 아래 요소를 중심 소재로 잡지 않는다.

```txt
유물
포션
저주 카드
상태 카드
전투 중 카드 생성
4막 / 진엔딩 키
승천 난이도
일일 도전
리더보드
```

이 빈자리는 동료, 서약 전술, 유대 점수, 장비, 이벤트 여관, 증표가 채운다.

## 3. 세계관 방향

### 기본 세계관

마법 재난으로 거의 붕괴한 세계에서, 낡은 현실형 용병이 죽은 지휘관의 증표와 작전서를 들고 버려진 고대 성의 코어를 부수러 가는 생존 판타지 로드무비.

```txt
장르 정서: 괴물 재난 속 로드무비 + 생존 판타지 + 전술 부대
배경 키워드: 마법 재난, 버려진 고대 성, 고대 마법 코어, 증표, 피 지문, 검은 지문, 위험한 여관, 동료의 흔적
주인공 설정: 발굴과 무관한 낡은 현실형 용병
```

세계관은 긴 컷신보다 배경, 적 디자인, 장비명, 이벤트 한두 줄, UI 소품으로 전달한다. 이 게임의 중심은 여전히 카드, 동료, 서약 전술, 유대 점수, 장비다.

세계관 세부 시각 기준도 이 문서 안에서 함께 관리한다. 아트디렉터용 문서는 이 파일 하나만 보면 된다.

```txt
재난 원인: 왕립 학회와 상단이 버려진 고대 성의 코어를 작동시킴.
현재 상태: 왕국 지휘 체계, 왕립 학회, 상단 모두 사실상 붕괴.
주인공 목표: 죽은 왕국 지휘관의 작전서를 따라 코어를 파괴.
증표: 피로 지문을 찍는 금속패 계약 토큰. 약속을 못 지키면 지문이 검게 변함.
괴물: 초반은 인간/동식물 변이체, 후반은 형체를 알 수 없는 이계 존재.
```

### 유지해야 할 정서

```txt
낡았지만 살아 있는 세계
위험하지만 완전히 절망적이지 않은 여정
반짝이는 왕궁보다 먼지 묻은 길과 발굴 흔적
차가운 청회색 길 위에 남은 모닥불의 온기
계약으로 시작해 신뢰로 바뀌는 동료 관계
```

### 피해야 할 정서

```txt
현대적인 네온 판타지
과도하게 화려한 모바일 수집형 RPG 느낌
너무 귀엽거나 장난감 같은 SD 비율
피와 공포를 앞세우는 다크 호러
캐릭터보다 장비와 이펙트만 튀는 그림
상단/학회 인물극이 중심이 되는 스토리 게임
```

### 3.1 세계의 현재 상태

왕국의 지휘 체계는 붕괴했다. 왕립 학회와 상단도 사실상 제 기능을 잃었다. 재난 이전의 권력 구조는 남아 있지만, 길 위에서 실질적으로 통하는 것은 물자, 무기, 잠자리, 그리고 증표뿐이다.

```txt
세계 상태: 마법 재난으로 거의 붕괴 중
재난 원인: 왕립 학회와 상단의 고대 유적 발굴
발원지: 다가가기 힘든 버려진 고대 성
최종 목표: 성 내부의 고대 마법 코어 파괴
주요 위협: 인간/동물/식물과 다른 차원의 존재가 섞인 괴물
```

재난의 기원:

```txt
왕립 학회와 상단이 버려진 고대 성을 발굴했다.
명분은 연구와 탐사였지만, 실제 목적은 부와 권력의 확보였다.
부패한 학회 고위층과 상단이 작동하면 안 되는 고대 마법 코어를 작동시켰다.
코어의 제어가 풀리며 다른 차원의 존재가 현실로 새어 나왔다.
```

상단은 현재 거의 괴멸했다. 초반부에는 살아 있는 조직 악역보다, 변이된 상단원이나 버려진 거래소 흔적으로 등장하는 정도가 적합하다.

### 3.2 주인공의 세계관 위치

주인공은 발굴과 무관한 용병이다. 사건의 책임자가 아니며, 세계를 구하겠다고 출발한 성인도 아니다.

```txt
출발점: 생존
전환점: 재난을 수습하려다 죽은 왕국 지휘관의 시체 발견
획득 물건: 지휘관의 증표, 작전서
알게 된 사실: 지금 코어를 부수지 않으면 세계가 완전히 무너진다
목표: 버려진 고대 성으로 가서 고대 마법 코어를 파괴한다
```

주인공은 세계의 중심 인물이 아니라, 망해가는 길 위에서 어쩔 수 없이 중심으로 걸어 들어간 사람이다.

### 3.3 증표 규칙

증표는 이 세계에서 중요한 문화적/법적 계약 토큰이다. 상단만의 물건이 아니며, 용병 계약, 통행 보증, 물자 거래, 호위 계약, 동료 합류, 위험 부담 합의에 널리 쓰인다.

```txt
이름: 증표
재질: 금속패
공증 기관: 없음
계약 방식: 제안자와 승낙자가 증표에 피로 지문을 찍고 맹세
소유자: 계약 조건에 따라 달라짐
마법 효력: 약속을 지키지 못하면 해당 지문이 검게 변함
판정: 사정과 무관하게 결과 기준
사회적 대가: 신용 상실, 거래 거부, 추방, 죽어 마땅한 자 취급
```

증표는 거짓말을 하지 않지만, 계약이 공정하다는 보장은 하지 않는다. 절박한 상황에서 한쪽에게 매우 불리한 조건이 걸릴 수 있고, 말장난 같은 계약 문구로 속이는 일도 가능하다. 억울하게 검은 지문을 갖게 된 사람도 있다.

게임 시스템과 연결:

```txt
동료 영입: 이번 여정의 증표를 함께 찍는 행위
서약 전술: 증표에 맺은 이번 런의 전투 조항
유대 점수: 계약이 인간적 신뢰로 바뀌는 정도
30/60/100 보너스: 증표 주변 장식, 피 지문, 끈, 흠집의 시각 변화
검은 지문 NPC: 동료 후보, 상점/여관 거래 거부 NPC, 이벤트 소재
```

아트 표현 원칙:

```txt
증표는 깨끗한 배지가 아니라 긁히고 닳은 금속패다.
피 지문은 작지만 즉시 읽혀야 한다.
검은 지문은 악인 확정 표식이 아니라 사회적 낙인이다.
서약 전술 발동은 큰 마법진보다 증표, 지문, 짧은 선, 금속 반응으로 표현한다.
```

### 3.4 괴물 디자인 진행 규칙

괴물은 원래 인간/동물/식물이 다른 차원의 존재와 섞인 결과다. 진행될수록 알아볼 수 있는 현실의 형태가 줄고, 다른 차원의 존재가 원형에 가깝게 나타난다.

| 구간 | 괴물 방향 | 아트 핵심 |
|---|---|---|
| Act 1 | 상단원, 왕립 학자, 용병, 늑대/말/새/식물 변이체 | 익숙한 것이 뒤틀린 공포. 원래 무엇이었는지 알아볼 수 있어야 함 |
| Act 2 | 탐사 장비, 금속패, 가방, 안경, 도구가 몸과 섞인 변이체 | 인간/동식물/장비 구조가 뒤섞이고 공간 왜곡 흔적이 보임 |
| Act 3 | 인간 실루엣이 무너진 이계 생물, 코어와 이어진 존재 | 괴물보다 차원의 규칙이 적으로 나타나는 느낌 |

### 3.5 주요 세력과 흔적

이 게임은 세력극이 중심이 아니다. 아래 세력은 인물 드라마보다 적, 배경, 소품, 이벤트 소재로 사용한다.

| 세력/흔적 | 역할 | 아트 작업 |
|---|---|---|
| 왕국 잔군 | 지휘 체계가 붕괴한 군대의 흔적 | 죽은 지휘관의 장비, 초소, 낡은 명령서, 붕괴한 군기 |
| 왕립 학회 잔재 | 고대 성 발굴을 주도한 연구 조직 | 학회 문서, 발굴 도구, 깨진 안경, 실험 장비, 변이 학자 |
| 상단 잔재 | 부와 권력욕으로 재난을 키운 세력 | 찢어진 장부, 금고, 물자 상자, 변이 상단원, 거래소 폐허 |
| 용병과 길 위의 사람들 | 주인공과 동료 대부분이 속한 생존자 층 | 낡은 무기, 수선된 갑옷, 증표 끈, 임시 야영지, 여관 손님 |
| 검은 지문을 가진 사람들 | 증표 계약을 지키지 못한 사회적 낙인자 | 검게 변한 지문, 가려진 손, 거래 거부 장면, 동료 후보의 낙인 |

### 3.6 동료들의 세계관 위치

동료는 거대한 메인 스토리의 주역이 아니라, 길 위에서 증표를 함께 찍는 계약자다. 각 동료는 전투 역할과 증표 조항이 바로 연결되어야 한다.

| 동료 | 세계관 위치 | 시각 단서 |
|---|---|---|
| 로완 | 표적 추격에 능한 용병 | 붉은 천, 긴 창, 낡은 군장 |
| 세라 | 길 위의 척후/암호 전달자 | 얇은 단검, 암호 표식, 검은 가죽 |
| 브람 | 발굴 장비와 폭약을 다루던 기술자 | 화약 주머니, 황동, 그을음 |
| 엘드릭 | 무너진 군 체계에서 살아남은 방패병 | 낡은 방패, 금속 보강, 중갑 |
| 마렌 | 재난 생존자를 돌보는 순례 치료사 | 회색 천, 약초, 낮은 빛 |
| 토르 | 외곽 방어선 출신 전사 | 두꺼운 가죽, 무거운 실루엣, 차가운 철 |
| 리나 | 오염된 숲길을 아는 약초술사 | 초록 병, 말린 잎, 가죽 가방 |
| 노아 | 학회 기록을 훔쳐 읽은 떠돌이 기록자 | 남색 천, 은색 잉크, 접힌 카드 |
| 이솔 | 무너진 길 위의 보호자 | 흰 천, 작은 은종, 보호 표식 |
| 카일 | 죽은 상단의 규칙을 이용하는 거래꾼 | 동전 주머니, 저울 흔적, 짙은 천 |

동료별 서약 전술 아이콘은 "멋진 문장"보다 **증표에 적힌 전투 조항**처럼 보여야 한다. 금속패, 피 지문, 긁힌 선, 짧은 계약 문구, 동료의 도구가 조합된 형태를 우선한다.

## 4. 비주얼 디자인 기둥

### 4.1 전술적 가독성

전투는 카드, 적 의도, 동료 기본 공격, 전술 표식이 동시에 보인다. 화면이 아름다워도 읽히지 않으면 실패다.

```txt
적 실루엣은 역할별로 분명해야 한다.
동료는 전투 화면에서 작은 초상만으로도 구분되어야 한다.
상태 효과와 의도 아이콘은 텍스트 없이도 1차 의미가 보여야 한다.
전술 표식은 턴 중 가장 중요한 시선 유도 장치다.
```

### 4.2 동료 중심의 런 정체성

동료는 숫자 보너스가 아니라 "이번 런의 빌드 문"이다. 영입 시 선택한 서약 전술은 증표에 맺은 전투 조항처럼 화면에 남겨야 한다.

```txt
로완 + 붉은 추격: 표식, 창끝, 붉은 지휘선.
세라 + 검은매 연계: 빠른 잔상, 얇은 단검, 암호 표식.
엘드릭 + 철벽 반격: 방패, 충격파, 긁힌 증표.
```

동료마다 실루엣, 주 색, 무기, 장비 궁합, 카드 프레임 포인트가 달라야 한다.

### 4.3 길 위의 물성

세계는 손에 잡히는 재료로 구성된다.

```txt
낡은 금속
해진 천
가죽 끈
목재 간판
양피지 지도
피 지문이 찍힌 증표
찌그러진 동전
그을린 돌
약초와 유리병
```

UI도 완전한 현대 평면 UI보다는, 전술 지도/카드/장비함의 물성을 얇게 품는 쪽이 어울린다. 단, 장식이 텍스트 가독성을 해치면 안 된다.

### 4.4 따뜻한 위험

어두운 판타지이되, 화면 전체가 검거나 칙칙하면 안 된다. 위험한 지역 안에도 불빛, 천 색, 금속 반사, 마법 색 같은 시선 포인트가 있어야 한다.

## 5. 전체 아트 스타일 제안

### 추천 방향

```txt
2D 일러스트 기반
반실사보다 스타일라이즈된 중세 판타지
선과 실루엣이 분명한 캐릭터
카드 일러스트는 회화적이되 작은 크기에서도 읽히는 구성
UI는 장식보다 정보 계층 우선
```

캐릭터 비율은 현실적인 6.5~7.5등신을 기본으로 한다. 너무 실사적이면 카드 게임의 조작감이 무거워지고, 너무 SD에 가까우면 로드 판타지의 생존감이 약해진다.

### 색채 원칙

전체 게임이 한 가지 색만으로 보이면 안 된다. Act별로 주조색은 다르되, 항상 보조색과 시선 유도색을 함께 둔다.

| 구간 | 기본 인상 | 주 색감 | 보조/강조 |
|---|---|---|---|
| Act 1 붕괴한 외곽길 | 재난 직후의 길과 임시 거처 | 차가운 청회색, 낡은 철, 먼지 낀 목재 | 따뜻한 모닥불색, 마른 피 색 |
| Act 2 뒤틀린 탐사로 | 폐광, 숲, 학회/상단 잔해 | 청회색, 탁한 녹색, 병든 황색 | 금속 회색, 들어오기 시작하는 어두운 보라 |
| Act 3 버려진 고대 성 | 인간 성에서 이계 공간으로 변질 | 어두운 보라, 진녹색, 검게 식은 금속 | 비현실적 코어 광원 |
| UI 공통 | 전술 지도와 증표/장비함 | 양피지, 먹색, 금속 회색 | 역할별 색상, 피 지문, 상태 아이콘 |

## 6. 화면 구성 방향

### 전투 화면

권장 구도:

```txt
좌측: 주인공과 보유 동료
우측: 적
하단: 손패 카드
상단 또는 측면: 체력, 골드, 남은 에너지, 덱/버림/소멸 더미
적 위: 의도 아이콘
대상 위: 전술 표식 아이콘
```

동료는 별도 체력이 없지만 화면에 존재해야 한다. 초상화만으로 끝나면 동료 시스템의 감정적 힘이 약해진다.

가능한 표현:

```txt
전투 좌측에 작은 스탠딩 실루엣 배치
하단 또는 좌측에 동료 초상 + 선택한 서약 전술 아이콘 + 유대 점수 게이지 + 장비 슬롯 요약
턴 종료 후 동료 기본 공격 시 짧은 컷인 또는 실루엣 액션
```

### 맵 화면

맵은 갈림길이 있는 로드 판타지의 핵심 화면이다.

```txt
양피지 또는 전술 지도 느낌
노드는 길 위의 장소 아이콘으로 표현
보스 방향은 항상 시각적으로 압박감이 있어야 함
Act 1 중반에는 동료의 흔적 노드가 특별히 눈에 띄어야 함
```

### 상점 화면

상점은 안전한 메뉴가 아니라, 길 위의 낡은 거래 장소다.

```txt
낡은 카운터
걸린 장비
카드 묶음
서비스 표지판
상인의 손과 동전
```

상점 UI 목표는 "전부 사고 싶지만 하나나 둘만 살 수 있다"는 경제 압박을 보여주는 것이다.

### 여관 화면

여관은 회복 노드이면서 이벤트 공간이다. 일반 여관과 이벤트 여관이 첫눈에 구분되어야 한다.

```txt
일반 여관: 정직하고 따뜻함, 가격과 효과 명확
이벤트 여관: 수상하지만 치명적 함정은 아님, 이상한 방 이름과 묘한 조명
```

이벤트 여관은 공포가 아니라 "수상한데 혹시 좋은 일이 생길지도 모른다"는 쪽이다.

## 7. 주인공 아트 방향

주인공은 동료 시스템을 받쳐주는 균형형 용병이다. 너무 튀면 동료가 흐려지고, 너무 밋밋하면 전투 화면의 중심이 약해진다.

```txt
역할: 발굴과 무관한 낡은 현실형 용병
무기: 검, 짧은 창, 한손 도끼 등 실용 무기 중 택1 권장
실루엣: 망토 또는 반망토, 실용적인 갑옷, 작전서/지도 소지
성격: 침착함, 피로감, 책임감
색감: 중립 금속색 + 낡은 천 + 피 지문이 찍힌 증표
```

피해야 할 것:

```txt
왕관을 쓴 영웅왕
과도하게 거대한 무기
동료보다 화려한 갑옷
사건의 책임자처럼 보이는 학자/귀족
```

## 8. 동료 10명 아트 방향

동료는 각자 전투 역할, 성격, 장비 궁합이 보이는 캐릭터여야 한다.

| 동료 | 역할 | 비주얼 핵심 | 색/소재 | 카드/VFX 힌트 |
|---|---|---|---|---|
| 붉은창 로완 | 전술 표식 단일 딜러 | 붉은 망토, 긴 창, 표적을 보는 자세 | 진홍 천, 어두운 철, 깃발 | 직선 찌르기, 붉은 표식, 취약 균열 |
| 검은매 세라 | 0비용 연타 암살자 | 짧은 검 두 자루, 가벼운 척후병 복장 | 검은 가죽, 청록 포인트, 은색 칼날 | 빠른 잔상, 얇은 베기 선, 암호 표식 |
| 화약장이 브람 | 광역 폭발 기술자 | 폭약 주머니, 그을린 장갑, 낮고 단단한 체형 | 그을음, 황동, 주황 불씨 | 폭발 원형, 파편, 연막 |
| 방패병 엘드릭 | 안정 방어 | 낡은 방패, 중갑, 곧은 자세 | 회색 철, 낡은 금장, 푸른 천 | 방패광, 반격 충격파, 긁힌 증표 |
| 수도승 마렌 | 회복/지속 생존 | 회색 수도복, 철제 지팡이, 약초 주머니 | 회색 천, 말린 약초, 따뜻한 빛 | 작은 기도문, 낮은 회복광, 호흡 이펙트 |
| 철갑 곰지기 토르 | 피해 흡수/약화 | 큰 가죽 갑옷, 북방 전사, 무거운 체중감 | 짙은 가죽, 털, 차가운 철 | 둔탁한 충격, 서리 입김, 약화 포효 |
| 독초술사 리나 | 독/디버프 | 초록 망토, 유리병, 약초 가방 | 초록 유리, 갈색 가죽, 노란 독액 | 독 방울, 연무, 부식 흔적 |
| 별읽는 노아 | 드로우/손패 조작 | 별무늬 로브, 카드/천문 도구 | 남색 천, 은색 별, 잉크 | 별선, 카드 궤도, 예지 원 |
| 백사제 이솔 | 보호/위기 회복 | 흰 사제복, 작은 은종, 성스러운 천 | 백색 천, 은, 따뜻한 금빛 | 종 파동, 보호막, 부드러운 회복광 |
| 은주머니 카일 | 골드/상점 보상 | 웃는 상인, 동전 주머니, 겹겹의 옷 | 노란 금속, 짙은 초록 천, 갈색 가죽 | 동전 튐, 거래 도장, 계산서 |

각 동료는 서약 전술 3개를 가진다. 아트에서는 3개 모두를 개별 컷신으로 만들 필요는 없지만, 적어도 **서약 전술 아이콘 3개**와 선택된 전투 조항을 보여주는 작은 증표 마커가 필요하다.

### MVP 우선 동료

첫 구현과 아트 테스트는 아래 3명으로 한다.

```txt
로완: 동료 시스템의 표식 조종감을 보여주는 기준 캐릭터
세라: 빠른 템포와 카드 회전의 재미를 보여주는 캐릭터
엘드릭: 생존형 동료도 매력적이어야 함을 증명하는 캐릭터
```

이 3명은 실루엣, 색상, 전투 감각이 명확히 달라야 한다.

## 9. 적과 보스 방향

### 적 공통 원칙

```txt
적은 역할이 실루엣으로 보여야 한다.
일반 적은 과하게 복잡하지 않게 만든다.
엘리트는 한눈에 위험도가 높아 보여야 한다.
보스는 Act의 테마를 시각적으로 대표한다.
```

적 의도와 애니메이션이 읽혀야 하므로, 장식이 너무 많거나 배경과 색이 붙으면 안 된다.

### Act 1: 붕괴한 외곽길

```txt
일반 적: 변이된 상단원, 변이된 왕립 학자, 변이된 용병, 뒤틀린 동식물
엘리트: 검은 지문을 가진 용병대장, 장비와 융합한 상단 경호원, 발굴 도구를 든 변이 학자
보스: 대형 변이 용병 또는 검은 지문을 가진 용병대장
```

Act 1 보스는 "동료를 만나기 전 마지막 혼자서의 시험"이다. 아직 인간이었음을 알아볼 수 있는 실루엣과 증표/장비 흔적이 중요하다.

### Act 2: 뒤틀린 탐사로

```txt
일반 적: 탐사대 변이체, 식물/동물 혼합 변이체, 학회 장비와 융합한 인간형
엘리트: 발굴 장비 변이체, 검은 지문 추방자, 약한 이계 생물
보스: 탐사 장비와 이계 생물이 결합한 대형 변이체
```

Act 2는 첫 동료와의 시너지가 본격화되는 구간이다. 적도 단일 해법보다 방어/공격/디버프 대응을 요구하는 형태로 보이면 좋다.

### Act 3: 버려진 고대 성

```txt
일반 적: 형태가 무너진 인간형, 성 내부에 남은 탐사대 변이체, 약한 이계 생물
엘리트: 현실 구조를 무시하는 이계 생물, 코어 주변 방어 장치, 고대 성에 융합된 변이체
보스: 고대 마법 코어 또는 코어와 결합한 존재
```

Act 3 보스는 주인공 덱, 동료 2명, 장비 조합의 최종 검증이다. 단순히 큰 괴물이 아니라, 재난의 발원지와 다른 차원의 규칙이 화면에 드러나는 존재여야 한다.

## 11. 카드와 UI 아트 방향

### 카드 프레임

카드는 이 게임의 가장 많이 보는 오브젝트다. 작은 화면에서도 비용, 타입, 희귀도, 소유자가 보여야 한다.

필수 구분:

```txt
소유자: 주인공 카드 / 동료 카드
타입: 공격 / 스킬 / 파워
희귀도: starter / common / uncommon / rare
비용: 좌상단 또는 가장 빠르게 읽히는 위치
강화 여부: 카드명 또는 프레임에 명확히 표시
```

동료 카드는 해당 동료의 색과 문양이 프레임 일부에 들어가야 한다. 단, 프레임이 너무 강하면 카드 일러스트와 텍스트가 묻힌다.

### 카드 일러스트

카드 일러스트는 전체 풀 기준으로 많다.

```txt
주인공 카드: 40장
동료 카드: 10명 x 8장 = 80장
```

초기에는 모든 카드에 풀 일러스트를 넣기보다, 카드 타입별/동료별 공통 모티프를 활용한 제작 효율안을 고려할 수 있다.

권장 접근:

```txt
MVP: 핵심 카드만 개별 일러스트, 나머지는 아이콘/모티프 기반 임시 아트 가능
정식: 주인공 주요 카드와 동료 카드 중심으로 개별 일러스트 확장
```

### 주요 UI 아이콘

```txt
에너지
체력
골드
방어도
카드 드로우
버림/소멸
전술 표식
취약
약화
독
회복
장비 슬롯: 투구 / 갑옷 / 무기
노드: 전투 / 엘리트 / 보스 / 상점 / 여관 / 이벤트 / 보물 / 동료 흔적
```

### 전술 표식

전술 표식은 동료 공격 대상을 조종하는 핵심 시각 요소다.

```txt
아이콘 후보: 작은 깃발, 화살표, 붉은 표적핀
표시 위치: 적 머리 위 또는 발밑 원형 마커
요구사항: 모바일에서도 즉시 보일 것
```

표식은 공격적이어야 하지만, 독/취약 같은 상태 효과와 혼동되면 안 된다.

### 서약 전술 아이콘

서약 전술은 동료 영입 시 3개 중 1개를 선택하는 고정 전투 규칙이다.

```txt
동료별 서약 전술 아이콘 3개
선택된 서약 전술 표시용 증표 마커
전투 중 발동 시 짧은 금속 반응/피 지문/선 연출
서약 전술은 업그레이드되지 않으므로 강화 단계 표현은 필요 없음
```

서약 전술 아이콘은 카드 타입 아이콘과 혼동되면 안 된다. 멋진 문장보다, 증표에 적힌 전투 조항과 동료의 도구가 결합된 형태가 좋다.

연출 강도는 카드의 주 효과보다 약간 낮게 잡는다. 서약 전술은 자주 발동될 수 있으므로, 매번 큰 컷신처럼 보이면 전투 리듬을 해친다.

### 유대 점수

동료 관계 성장은 `유대 점수 0~100`으로 표현한다.

```txt
30점: 친숙
60점: 신뢰
100점: 증표 완성
```

UI 표현 후보:

```txt
동료 초상 아래 얇은 유대 게이지
30/60/100 구간의 작은 증표 흠집/끈/지문 표시
100점 도달 시 증표 완성 연출
런 결과 화면의 전용 문구/작은 일러스트 포인트
```

유대 점수 보너스는 작은 관계 성장이다. 30/60/100점에서 캐릭터가 완전히 변신하는 표현보다는, 증표 주변의 끈, 피 지문, 흠집이 정돈되는 정도가 적합하다.

## 12. 장비 아트 방향

장비는 무제한 수집품이 아니다. 누가 착용하느냐가 중요하다.

장비 슬롯:

```txt
투구
갑옷
무기
```

캐릭터별 최대 3슬롯이며, 주인공 + 동료 2명 기준 최대 9슬롯이다.

장비 아이콘은 아래를 만족해야 한다.

```txt
작은 크기에서도 슬롯이 구분된다.
장비 희귀도가 보인다.
팀 전체 효과와 장착자 전용 효과가 구분된다.
동료 궁합을 암시하는 소재나 색이 있다.
```

장비 예시:

```txt
선봉대 철투구: 모든 동료 기본 공격 피해 +1
전술가의 모자: 주인공 첫 턴 드로우 +1
치유사의 은관: 회복 효과 +20%
지휘검: 모든 공격 카드 피해 +1
낡은 거래 증표: 상점 카드 가격 10% 감소
수상한 열쇠: 이벤트 선택지에서 보물 결과 확률 증가
```

## 13. VFX와 모션 방향

전투 애니메이션은 화려함보다 리듬과 판독성이 우선이다.

### 기본 원칙

```txt
카드 사용 후 0.2~0.5초 안에 결과가 읽혀야 한다.
동료 기본 공격은 자동으로 발생하므로 짧고 명확해야 한다.

| 약화 | 무거운 회색 보라, 흐려지는 실루엣 |
| 취약 | 붉은 균열, 깨지는 방패선 |
| 드로우/예지 | 잉크선, 별점, 종이 넘김 |
| 골드 | 작고 짧은 동전 반짝임 |
| 전술 표식 | 깃발/핀 고정, 짧은 지휘선 |

1. 증표의 기본 형태와 피 지문/검은 지문의 표현 방식
2. 버려진 고대 성과 코어의 키 비주얼 톤
3. 초반 인간/동식물 변이체가 원형을 얼마나 남길지
4. 캐릭터 비율과 렌더링 밀도
5. 카드 일러스트를 전부 개별 제작할지, 모티프/아이콘 혼합으로 갈지
6. 서약 전술 아이콘을 증표 조항형, 액션형, 카드형 중 어떤 방향으로 갈지
7. 전투 화면에서 동료를 스탠딩으로 보여줄지, 초상 중심으로 보여줄지
8. UI 물성의 정도: 양피지/금속 장식과 현대적 가독성의 비율
9. Act별 팔레트와 키 비주얼
10. MVP 기준으로 어디까지 완성 아트가 필요하고 어디까지 임시 아트가 허용되는지
```

가장 중요한 첫 결정은 **증표를 어떤 시각 언어로 표현할 것인가**다. 이 답이 서약 전술 아이콘, 유대 점수, 동료 영입 UI, 이벤트 소품, 검은 지문 NPC를 모두 묶는다. 그다음 결정은 동료가 전투 화면에서 얼마나 물리적으로 존재하는가이며, 이 결정이 캐릭터 제작량, 애니메이션, UI 레이아웃, 카드 보상 화면의 감정선을 좌우한다.
