# 아트디렉터용 세계관 브리프

작성일: 2026-05-26  
목적: 게임의 세계관을 시각 작업 가능한 수준으로 정리하고, 아트디렉터가 우선 제작해야 할 콘셉트/키아트/에셋 항목을 정의한다.

---

## 1. 세계관 한 줄

**마법 재난으로 거의 붕괴한 세계에서, 낡은 현실형 용병이 죽은 지휘관의 증표와 작전서를 들고 버려진 고대 성의 코어를 부수러 가는 생존 판타지 로드무비.**

이 게임은 스토리텔링형 게임이 아니다. 세계관은 긴 컷신이나 대사극을 위한 장치가 아니라, 카드/동료/서약 전술/유대 점수/장비/이벤트를 한 시각 언어로 묶기 위한 기준이다.

```txt
핵심 경험: 카드 전투 + 동료 계약 + 증표 + 장비 + 위험한 길 위의 선택
세계관 전달 방식: 배경, 적 디자인, 장비명, 이벤트 한두 줄, UI 소품
피해야 할 방식: 긴 메인 퀘스트 대사, 복잡한 인물 관계극, 컷신 중심 진행
```

## 2. 핵심 정서

```txt
괴물 재난 속 로드무비
생존 판타지
낡은 현실형 용병
피로 찍은 증표
계약으로 시작해 신뢰로 바뀌는 동료 관계
차가운 길과 따뜻한 임시 거처
깊이 갈수록 현실 법칙이 흐려지는 고대 성
```

### 보여줘야 하는 감각

```txt
세계는 이미 끝난 것이 아니라 거의 다 망해가는 중이다.
사람들은 아직 거래하고, 쉬고, 싸우고, 증표를 믿는다.
주인공은 운명적 영웅이 아니라 살아남는 법을 아는 용병이다.
동료 관계는 감상적인 운명보다 조건이 있는 계약에서 출발한다.
유대는 계약 조건을 넘어서 서로를 믿게 되는 과정이다.
```

### 피해야 하는 감각

```txt
찬란한 왕도 중심 영웅담
선악이 또렷한 기사단 판타지
상단/학회 인물극이 중심이 되는 스토리 게임
전체가 검정/보라만으로 읽히는 다크 판타지
캐릭터 수집형 RPG처럼 지나치게 화려한 복식
```

## 3. 세계의 현재 상태

왕국의 지휘 체계는 붕괴했다. 왕립 학회와 상단도 사실상 제 기능을 잃었다. 재난 이전의 권력 구조는 남아 있지만, 길 위에서 실질적으로 통하는 것은 물자, 무기, 잠자리, 그리고 증표뿐이다.

```txt
세계 상태: 마법 재난으로 거의 붕괴 중
재난 원인: 왕립 학회와 상단의 고대 유적 발굴
발원지: 다가가기 힘든 버려진 고대 성
최종 목표: 성 내부의 고대 마법 코어 파괴
주요 위협: 인간/동물/식물과 다른 차원의 존재가 섞인 괴물
```

### 재난의 기원

```txt
왕립 학회와 상단이 버려진 고대 성을 발굴했다.
명분은 연구와 탐사였지만, 실제 목적은 부와 권력의 확보였다.
부패한 학회 고위층과 상단이 작동하면 안 되는 고대 마법 코어를 작동시켰다.
코어의 제어가 풀리며 다른 차원의 존재가 현실로 새어 나왔다.
```

상단은 현재 거의 괴멸했다. 초반부에는 살아 있는 조직 악역보다, 변이된 상단원이나 버려진 거래소 흔적으로 등장하는 정도가 적합하다.

## 4. 주인공의 위치

주인공은 발굴과 무관한 용병이다. 사건의 책임자가 아니며, 세계를 구하겠다고 출발한 성인도 아니다.

```txt
출발점: 생존
전환점: 재난을 수습하려다 죽은 왕국 지휘관의 시체 발견
획득 물건: 지휘관의 증표, 작전서
알게 된 사실: 지금 코어를 부수지 않으면 세계가 완전히 무너진다
목표: 버려진 고대 성으로 가서 고대 마법 코어를 파괴한다
```

### 주인공 아트 기준

```txt
낡은 현실형 용병
실용적인 갑옷과 해진 망토
과장되지 않은 무기
작전서 또는 접힌 지도
피와 흠집이 남은 증표
화려한 왕가 혈통처럼 보이지 않을 것
```

주인공은 세계의 중심 인물이 아니라, 망해가는 길 위에서 어쩔 수 없이 중심으로 걸어 들어간 사람이다.

## 5. 증표

증표는 이 세계에서 중요한 문화적/법적 계약 토큰이다. 상단만의 물건이 아니며, 용병 계약, 통행 보증, 물자 거래, 호위 계약, 동료 합류, 위험 부담 합의에 널리 쓰인다.

```txt
이름: 증표
재질: 금속패
공증 기관: 없음
계약 방식: 제안자와 승낙자가 증표에 피로 지문을 찍고 맹세
소유자: 계약 조건에 따라 달라짐
마법 효력: 약속을 지키지 못하면 해당 지문이 검게 변함
판정: 사정과 무관하게 결과 기준
사회적 대가: 신용 상실, 거래 거부, 추방, 죽어 마땅한 자 취급
```

증표는 거짓말을 하지 않지만, 계약이 공정하다는 보장은 하지 않는다. 절박한 상황에서 한쪽에게 매우 불리한 조건이 걸릴 수 있고, 말장난 같은 계약 문구로 속이는 일도 가능하다. 억울하게 검은 지문을 갖게 된 사람도 있다.

### 게임 시스템과 연결

```txt
동료 영입: 이번 여정의 증표를 함께 찍는 행위
서약 전술: 증표에 맺은 이번 런의 전투 조항
유대 점수: 계약이 인간적 신뢰로 바뀌는 정도
30/60/100 보너스: 증표 주변 장식, 피 지문, 끈, 흠집의 시각 변화
검은 지문 NPC: 동료 후보, 상점/여관 거래 거부 NPC, 이벤트 소재
```

### 아트 표현 원칙

```txt
증표는 깨끗한 배지가 아니라 긁히고 닳은 금속패다.
피 지문은 작지만 즉시 읽혀야 한다.
검은 지문은 악인 확정 표식이 아니라 사회적 낙인이다.
서약 전술 발동은 큰 마법진보다 증표, 지문, 짧은 선, 금속 반응으로 표현한다.
```

## 6. 괴물 디자인 진행 규칙

괴물은 원래 인간/동물/식물이 다른 차원의 존재와 섞인 결과다. 진행될수록 알아볼 수 있는 현실의 형태가 줄고, 다른 차원의 존재가 원형에 가깝게 나타난다.

### Act 1 방향

```txt
상단원 변이체
왕립 학자 변이체
용병 변이체
늑대/말/새/식물 변이체
인간 도구와 몸이 아직 구분되는 상태
```

Act 1은 익숙한 것이 뒤틀린 공포다. 적이 무엇이었는지 알아볼 수 있어야 한다.

### Act 2 방향

```txt
탐사 장비와 살이 섞임
금속패, 가방, 안경, 도구가 몸에 박힘
식물/동물/인간 구조가 뒤섞임
공간 왜곡의 흔적이 몸에 나타남
```

Act 2는 재난의 범위가 넓어지고, 길과 숲과 폐허가 같은 규칙으로 뒤틀리는 구간이다.

### Act 3 방향

```txt
인간 실루엣 붕괴
동식물 구조 붕괴
형체를 알 수 없는 이계 생물
고대 성 내부 공간과 이어진 듯한 몸
코어의 빛과 같은 색을 띠는 존재
```

Act 3은 괴물이 아니라 차원의 규칙이 적으로 나타나는 느낌이 필요하다.

## 7. Act별 세계관과 시각 방향

### Act 1. 붕괴한 외곽길

주인공은 아직 혼자다. 길 위에는 왕국과 상단과 학회의 흔적이 뒤섞여 있고, 초반 괴물은 대부분 알아볼 수 있는 변이체다.

```txt
주요 장소:
- 버려진 초소
- 부서진 마차
- 임시 용병 야영지
- 폐허가 된 거래소
- 작은 여관
- 녹슨 무기 상점
- 죽은 지휘관의 흔적이 남은 길목
```

```txt
적 방향:
- 변이된 상단원
- 변이된 왕립 학자
- 변이된 용병
- 뒤틀린 늑대/말/새
- 식물 변이체
```

```txt
색채:
- 차가운 청회색
- 낡은 철
- 먼지 낀 목재
- 따뜻한 모닥불색
- 마른 피 색
```

### Act 2. 뒤틀린 탐사로

첫 동료와의 전술이 본격화된다. 재난의 흔적은 길, 숲, 폐광, 폐허를 오염시키며 인간의 장비와 이계적 형태가 섞인다.

```txt
주요 장소:
- 폐광 입구
- 뒤틀린 숲길
- 오염된 강가
- 버려진 탐사 야영지
- 학회 임시 연구소 잔해
- 상단 물자 창고 폐허
- 증표 거래가 남은 여관
```

```txt
적 방향:
- 탐사대 변이체
- 장비와 융합한 인간형 변이체
- 식물/동물 혼합 변이체
- 검은 지문을 단 추방자
- 약한 이계 생물
```

```txt
색채:
- 차가운 청회색 유지
- 탁한 녹색
- 병든 황색
- 금속 회색
- 후반부로 들어오는 어두운 보라
```

### Act 3. 버려진 고대 성

최종 목적지는 버려진 고대 성이다. 겉은 인간이 지은 오래된 성처럼 보이지만, 내부로 갈수록 공간이 점점 비현실적으로 변한다.

```txt
주요 장소:
- 고대 성 외곽
- 무너진 성문
- 비정상적으로 긴 복도
- 방향 감각이 사라지는 계단
- 현실과 다른 차원이 겹친 홀
- 고대 마법 코어가 있는 중심부
```

```txt
적 방향:
- 성 내부에 남은 탐사대 변이체
- 형태가 무너진 인간형
- 현실 구조를 무시하는 이계 생물
- 코어 주변의 고대 방어 장치
- 최종 보스: 고대 마법 코어 또는 코어와 결합한 존재
```

```txt
색채:
- 어두운 보라
- 진녹색
- 검게 식은 금속
- 차가운 성벽 회색
- 비현실적 코어 광원
```

## 8. 주요 세력/흔적

이 게임은 세력극이 중심이 아니다. 아래 세력은 인물 드라마보다 적, 배경, 소품, 이벤트 소재로 사용한다.

### 왕국 잔군

왕국의 지휘 체계는 붕괴했지만, 일부 병사와 지휘관의 흔적은 남아 있다.

```txt
아트 작업: 죽은 지휘관의 장비, 초소, 낡은 명령서, 붕괴한 군기
```

### 왕립 학회 잔재

고대 성 발굴을 주도한 연구 조직. 현재는 생존자보다 변이체와 기록물로 더 많이 보인다.

```txt
아트 작업: 학회 문서, 발굴 도구, 깨진 안경, 실험 장비, 변이 학자
```

### 상단 잔재

부와 권력욕으로 재난을 키운 세력. 조직은 거의 죽었고, 초반부 변이체와 폐허가 된 물자 창고로 흔적을 남긴다.

```txt
아트 작업: 찢어진 장부, 금고, 물자 상자, 변이 상단원, 거래소 폐허
```

### 용병과 길 위의 사람들

주인공과 동료 대부분이 속한 현실적 생존자 층이다. 증표 문화가 가장 강하게 드러나는 집단이다.

```txt
아트 작업: 낡은 무기, 수선된 갑옷, 증표 끈, 임시 야영지, 여관 손님
```

### 검은 지문을 가진 사람들

증표 계약을 지키지 못해 지문이 검어진 사람들. 악인도 있지만, 불리한 계약이나 말장난에 당한 사람도 있다.

```txt
아트 작업: 검게 변한 지문, 가려진 손, 거래 거부 장면, 동료 후보의 낙인
```

## 9. 동료들의 세계관 위치

동료는 거대한 메인 스토리의 주역이 아니라, 길 위에서 증표를 함께 찍는 계약자다. 각 동료는 전투 역할과 증표 조항이 바로 연결되어야 한다.

| 동료 | 세계관 위치 | 시각 단서 |
|---|---|---|
| 로완 | 표적 추격에 능한 용병 | 붉은 천, 긴 창, 낡은 군장 |
| 세라 | 길 위의 척후/암호 전달자 | 얇은 단검, 암호 표식, 검은 가죽 |
| 브람 | 발굴 장비와 폭약을 다루던 기술자 | 화약 주머니, 황동, 그을음 |
| 엘드릭 | 무너진 군 체계에서 살아남은 방패병 | 낡은 방패, 금속 보강, 중갑 |
| 마렌 | 재난 생존자를 돌보는 순례 치료사 | 회색 천, 약초, 낮은 빛 |
| 토르 | 외곽 방어선 출신 전사 | 두꺼운 가죽, 무거운 실루엣, 차가운 철 |
| 리나 | 오염된 숲길을 아는 약초술사 | 초록 병, 말린 잎, 가죽 가방 |
| 노아 | 학회 기록을 훔쳐 읽은 떠돌이 기록자 | 남색 천, 은색 잉크, 접힌 카드 |
| 이솔 | 무너진 길 위의 보호자 | 흰 천, 작은 은종, 보호 표식 |
| 카일 | 죽은 상단의 규칙을 이용하는 거래꾼 | 동전 주머니, 저울 흔적, 짙은 천 |

동료별 서약 전술 아이콘은 "멋진 문장"보다 **증표에 적힌 전투 조항**처럼 보여야 한다. 금속패, 피 지문, 긁힌 선, 짧은 계약 문구, 동료의 도구가 조합된 형태를 우선한다.

## 10. 아트디렉터 우선 작업 목록

### 10.1 세계관 키 비주얼

```txt
1. 메인 키아트
   - 낡은 현실형 용병 주인공 + MVP 동료 3명 + 괴물 재난의 외곽길.
   - 분위기: 도망이 아니라 살아남기 위해 발원지로 향하는 여정.

2. 증표 키아트
   - 닳은 금속패, 피 지문, 검게 변한 지문, 끈, 흠집.
   - 서약 전술/유대 점수/UI의 공통 기준.

3. 버려진 고대 성 키아트
   - 겉은 인간 성, 내부는 비현실적 공간.
   - 코어의 광원과 Act 3 팔레트 기준.
```

### 10.2 Act별 컬러키

```txt
Act 1 붕괴한 외곽길: 3장
- 외곽길/작은 여관/폐허가 된 거래소.

Act 2 뒤틀린 탐사로: 3장
- 폐광/뒤틀린 숲길/학회 임시 연구소 잔해.

Act 3 버려진 고대 성: 3장
- 성 외곽/비현실적 복도/코어 중심부.
```

### 10.3 증표/소품 시트

```txt
일반 증표
피 지문이 찍힌 증표
검은 지문 증표
죽은 지휘관의 증표
작전서
학회 발굴 도구
상단 장부와 금고
여관 거래 표지
용병 장비 끈
고대 코어 파편
```

### 10.4 서약 전술 아이콘 방향

MVP 우선:

```txt
로완 서약 전술 아이콘 3개
세라 서약 전술 아이콘 3개
엘드릭 서약 전술 아이콘 3개
선택된 서약 표시용 증표 마커
서약 발동용 짧은 금속/피 지문 VFX 러프
```

전체:

```txt
동료 10명 x 서약 전술 3개 = 30개 아이콘
유대 점수 30/60/100 구간 표시
100점 증표 완성 표시
```

### 10.5 적/보스 콘셉트

MVP 우선:

```txt
Act 1 변이 상단원 2종
Act 1 변이 학자 2종
Act 1 변이 용병 2종
Act 1 동물/식물 변이체 2종
Act 1 보스: 검은 지문을 가진 용병대장 또는 대형 변이체
```

정식 1차:

```txt
Act 2 탐사대/장비 융합 변이체

Act 3 이계 생물
Act 3 고대 성 방어 장치
Act 3 보스: 고대 마법 코어 또는 코어와 결합한 존재
```

## 11. 첫 미팅에서 결정할 것

아트디렉터와 가장 먼저 결정해야 할 질문:

```txt
1. 증표의 기본 형태는 얼마나 투박하고 실용적인가?
2. 피 지문과 검은 지문은 작은 UI 크기에서 어떻게 구분하는가?
3. 고대 성의 내부가 비현실적으로 변하는 정도는 Act 3에서 어디까지 허용하는가?
4. 코어 광원은 어두운 보라와 진녹색 중 어느 쪽을 주 색으로 삼는가?
5. 초반 변이체는 얼마나 인간/동물 원형을 남길 것인가?
6. 동료 영입 화면에서 증표를 찍는 행위를 직접 보여줄 것인가, UI 마커로만 처리할 것인가?
7. 카드 일러스트는 개별 회화형인가, 증표/도구/행동 모티프 혼합형인가?
```

첫 결정의 중심은 **증표를 어떤 시각 언어로 표현할 것인가**다. 이 답이 서약 전술 아이콘, 유대 점수, 동료 영입 UI, 이벤트 소품, 검은 지문 NPC를 모두 묶는다.
